How to Install

For Windows Users

Download the Windows installer, then run the file. Follow the installer instructions to install the game. (This installation automatically comes packaged with Java 25.)

After it is installed, you can run the file TARN.exe from either the start menu or desktop shortcut to play the game.

For Users with Java

Install one of the following releases of Java.

Java 25+: The installer for the latest release can be found here. The latest release has minor performance improvements and is backwards-compatible with older releases, but Oracle requires you to register an account in order to download the file.

Java 8: The installer for the latest release of Java 8 can be found here and does not require an account. This release can only run the Java 8 executable of the game with minimal differences from the Java 25 version.

After Java is installed, download the Jar executable, then you can run the file TARN_<version>.jar to play the game.
